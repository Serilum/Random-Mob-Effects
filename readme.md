<p><span style="font-size:18px">Random Mob Effects is a configurable mod created to make the game a little more difficult. Every hostile mob receives a random potion effect upon joining the world. There are a few potion effects disabled by default, but a blacklist is created on first load and you may change the values. The effect applied has a duration of the maximum integer value in java, which is&nbsp;2147483647 ticks; around 30000 hours in-game. The particle effect can be disabled.<br></span><br><br><strong><span style="font-size:20px">Configurable:</span> <span style="color:#008000;font-size:14px"><a style="color:#008000" href="https://serilum.com/url/issue-trackerwiki/how-to-configure-mods" rel="nofollow">(&nbsp;how do I configure?&nbsp;)</a></span><br></strong><span style="font-size:12px"><strong>potionEffectLevel</strong>&nbsp;(default = 1, min 1, max 50): The default level of the effects the mod applies, by default 1.</span><br><span style="font-size:12px"><strong>hideEffectParticles</strong>&nbsp;(default = false): When enabled, hides the particles from the mobs with an effect.<br><strong>disableCreepers</strong>&nbsp;(default = true): Creepers can create infinite lingering potion effects which is probably not what you want. When enabled, the mod doesn't give creepers a random effect.</span><br><br><br><span style="font-size:14px"><span style="font-size:18px"><strong>The default blacklist</strong></span><br>is generated upon first load of the mod. It cycles through all potion effects within the Forge registry. This means that modded effects are supported, just make sure to refresh the config if a new mod with an effect is added. <em>You can disable an effect by adding an exclamation mark (!) in front of the name.</em><br>The blacklist file is located in <em><strong>./config/randommobeffects/</strong></em> and is called <em><strong>blacklist.txt</strong></em>.<br><picture><img src="https://github.com/Serilum/.cdn/raw/main/projects/random-mob-effects/a.png" width="1081" height="948"></picture></span><br><br><span style="font-size:18px"><strong>Two GIFs displaying the hostile mobs, each with their own potion effect:</strong></span></p>
